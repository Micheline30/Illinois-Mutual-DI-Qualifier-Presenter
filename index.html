    💼 Occupation Class Definitions
    ClassSIDIDescriptionCommon Examples
    Class 5AAdministrative, professional, managerial, clerical — no manual labor or service demandsAccountant, Attorney, Actuary, Computer Analyst, Secretary, Bookkeeper, Insurance Broker
    Class 4BProfessional/specialized technical — physical dexterity, little or no manual labor. Max issue $8K/mo.Surgeon, Dentist, CRNA, Oral Surgeon, Orthodontist, Endodontist, Cardiologist
    Class 3BSemi-professional, supervisory, or technical — on-site supervision, moderate dexterityDental Hygienist, Medical Technician, Physical Therapist, On-site Supervisor, RN (hospital)
    Class 2CSkilled trade — continual physical dexterity and manual labor/services, specialized training requiredCarpenter, Electrician, Farmer, Mechanic, Plumber, Cosmetologist, LPN, Paramedic
    Class 1CHeavy manual labor, hazardous exposure — basic skills, longer recuperation periods expectedChiropractor, Factory Laborer, Firefighter, Police Officer, Roofer, Carpet Installer, Welder
    NININot Insurable — severe hazard, extraordinary psychological stressors, extreme physical dexterityAir Traffic Controller, Pilot, Longshoreman, Armed Forces (all branches), SWAT, Bomb Squad
    📌 Key Rule: Occupation class is determined by actual duties performed, not job title alone. Obtain percentage of time in professional/managerial duties vs. manual labor. Class 4 and Chiropractors: max $8,000/month issue, $10,000/month participation. The Underwriting Department has final approval authority.
